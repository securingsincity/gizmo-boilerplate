# gizmo-boilerplate
NYTimes Gizmo Boilerplate

Gets you started with the gizmo microframework using MySQL/GORM.

## Install

With a properly configured Go toolchain:

`go get github.com/securingsincity/gizmo-boilerplate`
`go run main.go`
